gauMonitor v3.0.0 Update

Created on September 12, 2021 by Zhe Wang

2021   ·   Software   Computation

The Gaussian monitoring tool, gauMonitor, has been updated to version 3.0.0 on 12 Sep 2021. New user interface for improved readability and some new features have been added into gauMonitor v3.

New fetures in v3.0.0

Updated userinterface for SCF iteration section and energy section.
